# 👑 Coomer 佬友严选

> **OnlyFans 赛博菩萨，佬友严选，值得信赖！**
>
> 专为 Coomer 打造的沉浸式浏览体验增强脚本，提供艺术家收藏、作品管理、视频播放增强及全站 UI 重构。

![Version](https://img.shields.io/badge/version-1.0.0-blue.svg) ![License](https://img.shields.io/badge/license-MIT-green.svg)

## ✨ 主要功能

### 1. 🎨 黑金极简 UI (Obsidian & Gold)

-   **视觉重构**：全站采用沉浸式黑金配色 (`#121212` + `#E0AA3E`)，去除原有紫色调。
-   **Modern Design**：Glassmorphism 磨砂质感面板，细腻的卡片阴影与微动效。
-   **极简滚动条**：定制 Webkit 滚动条，悬停流金效果，丝般顺滑。

### 2. 👩‍🎨 艺术家与作品管理

-   **一键收藏**：通过悬浮球或快捷按钮，轻松收藏喜欢的艺术家和作品。
-   **自定义分组**：
    -   **📌 佬友严选**：内置高质量预置艺术家推荐。
    -   **⭐ 我的收藏**：管理你自己收藏的宝藏艺术家。
-   **作品管理**：收藏特定作品，支持置顶、删除操作。

### 3. 🎬 增强型视频播放器

-   **内置播放**：彻底告别 "打开新标签页下载" 的糟糕体验，直接在页面内弹窗播放。
-   **Video.js 深度集成**：
    -   **倍速播放**：0.5x - 2.0x 自由切换。
    -   **快进/快退**：支持 15s 快速跳转与键盘快捷键控制。
    -   **画中画 & 全屏**：支持双击全屏、画中画模式。
-   **自定义主题**：播放器 UI 完美适配黑金主题。

### 4. 🚀 体验优化

-   **SPA 无刷新导航**：优化页面跳转逻辑，减少白屏等待。
-   **快捷操作**：智能悬浮快捷按钮，随手收藏/取消收藏。
-   **媒体数量角标**：直观显示帖子内的图片/视频数量。

## 📦 安装指南

1. 安装油猴管理器：[Tampermonkey](https://www.tampermonkey.net/)
2. 安装脚本：[点击安装](https://greasyfork.org/zh-CN/scripts/558318-coomer) _(链接仅为示例)_
3. 访问 [Coomer.st](https://coomer.st/) 即可生效。

## 📖 使用说明

1. **呼出面板**：

    - 页面右下角会出现一个带有金色光环的**黑金悬浮球**。
    - 点击悬浮球即可打开/关闭「佬友严选」管理面板。
    - 悬浮球支持拖拽至屏幕任意位置。

2. **视频播放**：

    - 点击任意视频帖子的附件链接，不再是下载，而是直接弹出播放器播放。
    - 使用播放器底部的控制条或快捷键进行控制。

3. **收藏/管理**：
    - 在面板的「艺术家」或「作品」Tab 中管理你的收藏。
    - 在浏览页面时，鼠标悬停在悬浮球上会弹出快捷操作按钮，可快速收藏当前页面。
