> Gemini 多功能助手：提示词管理、页面加宽、快捷导航、多语言支持

**主要功能**：

- ✅ 提示词管理（快速插入、分类、搜索、拖动排序）
- ✅ 页面加宽（支持 px/% 单位，即时生效）
- ✅ 快捷导航（跳转顶部/底部）
- ✅ 设置面板（语言切换、中文输入修复）
- ✅ 多站点支持（Gemini 标准版/企业版、Genspark）

**最新版本**: v1.6.1
