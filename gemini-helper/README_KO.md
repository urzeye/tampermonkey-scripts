# Gemini-helper

> Gemini 도우미: 대화 관리 및 내보내기, 개요 탐색, 프롬프트 관리, 탭 향상(상태/개인정보/알림), 읽기 기록 및 복원, 양방향/수동 앵커, 이미지 워터마크 제거, 굵게 수정, 수식/표 복사, 모델 잠금, 페이지 미화, 테마 전환, 스마트 다크 모드(Gemini/Gemini Enterprise 지원)

🌐 **언어**: [简体中文](README.md) | [English](README_EN.md) | [日本語](README_JA.md) | **한국어** | [Deutsch](README_DE.md) | [Français](README_FR.md) | [Español](README_ES.md) | [Português](README_PT.md) | [Русский](README_RU.md)

## ✨ 기능

### 📝 프롬프트 관리

- **빠른 삽입**: 자주 사용하는 프롬프트를 원클릭으로 채팅에 삽입
- **카테고리 관리**: 카테고리 필터, 이름 변경, 삭제 지원
- **검색 기능**: 필요한 프롬프트를 빠르게 검색
- **CRUD 작업**: 프롬프트 라이브러리 사용자 정의 및 관리
- **복사 기능**: 프롬프트 내용을 원클릭으로 클립보드에 복사
- **드래그 앤 정렬**: 프롬프트 표시 순서를 자유롭게 조정

### 📁 대화 관리

- **폴더 아카이브**: 사용자 정의 폴더를 만들어 채팅 기록 정리
- **다색 태그**: 30가지 이상의 전통 중국 색상, 사용자 정의 색상 및 다중 태그 관리 지원
- **실시간 검색**: 제목으로 빠른 필터, 태그 조합 필터링 지원
- **일괄 작업**: 일괄 삭제, 이동, 아카이브를 위한 다중 선택
- **대화 내보내기**: Markdown/JSON/TXT 형식으로 내보내기, 이미지는 Base64로 변환 가능
- **원활한 동기화**: Gemini 사이드바의 최신 데이터 자동 동기화(표준/엔터프라이즈 호환)

### 📑 개요 탐색

- **자동 추출**: AI 응답에서 제목 구조 추출(표준 및 엔터프라이즈 Shadow DOM 지원)
- **사용자 질문 그룹**: 대화 턴별로 개요 그룹화, 사용자 질문을 그룹 헤더로(💬 아이콘)
- **스마트 들여쓰기**: 최고 레벨에 따라 들여쓰기 자동 조정으로 왼쪽 공백 감소
- **빠른 이동**: 개요 항목 클릭으로 부드럽게 스크롤하고 2초간 강조 표시
- **동기화 스크롤**: 페이지 스크롤 시 해당 개요 항목 자동 강조(설정에서 전환 가능)
- **레벨 필터**: 제목 레벨 표시 설정, 레벨 0으로 사용자 질문만 빠르게 접기
- **전환 컨트롤**: 비활성화 시 개요 탭 자동 숨김

### 🚀 빠른 탐색

- **맨 위/아래로 이동**: 긴 대화에서 빠른 위치 지정
- **플로팅 버튼 그룹**: 패널 접힌 상태에서도 접근 가능

### 📐 페이지 너비

- **사용자 정의 너비**: 픽셀(px)과 퍼센트(%) 단위 모두 지원
- **즉시 적용**: 조정 후 즉시 적용, 새로고침 불필요
- **독립 설정**: 사이트별로 다른 설정 가능

### ⚓ 스마트 위치 시스템

두 가지 독립적인 위치 기록 시스템:

- **읽기 기록(Reading Progress)**:
  - 장기적인 "읽기 진행 기억", 새로고침/세션 간 복원 지원
  - 스크롤 시 자동 기록, GM_storage에 영구 저장
  - 페이지 로드 또는 대화 전환 시 자동 복원

- **양방향 앵커**:
  - 단기적인 "복귀 지점", 브라우저 뒤로 가기 또는 `git switch -`와 유사
  - 개요/맨 위/맨 아래 버튼 클릭 시 현재 위치 자동 저장
  - 두 위치 간 왕복 전환 지원

### 🏷️ 탭 향상

- **생성 상태 표시**: 탭 제목에 ⏳(생성 중) 또는 ✅(완료) 상태 아이콘 자동 표시
- **사용자 정의 제목 형식**: `{status}{title}[{model}]` 플레이스홀더 조합 지원
- **프라이버시 모드(보스 키)**: 원클릭으로 탭 제목을 "Google"로 위장, 대화 내용 숨김
- **완료 알림**: 백그라운드 생성 완료 시 데스크톱 알림 전송
- **자동 창 포커스**: 생성 완료 시 브라우저 창을 자동으로 전면으로

### ⚙️ 설정 패널

- **탭 전환**: 프롬프트, 개요, 설정 세 가지 탭
- **패널 설정**: 기본 확장/접기 사용자 정의, 외부 클릭 시 자동 숨김
- **중국어 입력 수정**: 엔터프라이즈에서 첫 글자 문제 수정 옵션 토글
- **언어 전환**: 간체 중국어/번체 중국어/영어 지원

### 🎯 스마트 적응

- ✅ Gemini 표준 (gemini.google.com)
- ✅ Gemini 엔터프라이즈 (business.gemini.google)

### 🌓 자동 다크 모드

- **스마트 감지**: 시스템/페이지 라이트/다크 모드 전환 실시간 추적
- **완전 적응**: 신중하게 조정된 다크 테마 색상 구성, 눈에 편안함

### 📋 콘텐츠 지원

- **수식 더블클릭 복사**: 수학 수식을 더블클릭하여 LaTeX 소스 복사, 구분자 자동 추가
- **표 Markdown 복사**: 표 오른쪽 상단에 복사 버튼 추가, Markdown 형식 직접 복사
- **워터마크 제거**: Gemini AI 생성 이미지에서 NanoBanana 워터마크 자동 제거
- **엣지 스냅 숨기기**: 패널을 화면 가장자리로 드래그하면 자동 숨김, 호버 시 표시
- **수동 앵커**: 빠른 도구 모음에서 앵커 위치 설정/복귀/지우기

## 📸 미리보기

- 오른쪽에 플로팅 패널, 드래그 앤 이동 지원(최적화된 경험, 실수로 텍스트 선택 없음)
- 그라데이션 테마, 아름다운 외관
- 플로팅 바에 현재 프롬프트 표시, 원클릭 지우기 지원

## 🔧 사용법

1. Tampermonkey 브라우저 확장 설치
2. 이 스크립트 설치
3. Gemini 페이지를 열면 오른쪽에 프롬프트 관리 패널이 나타납니다
4. 프롬프트를 클릭하여 빠르게 삽입
