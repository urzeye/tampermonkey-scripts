# Gemini-helper

> Gemini ヘルパー：会話管理とエクスポート、アウトラインナビゲーション、プロンプト管理、タブ拡張（ステータス/プライバシー/通知）、閲覧履歴と復元、双方向/手動アンカー、画像透かし除去、太字修正、数式/テーブルコピー、モデルロック、ページ美化、テーマ切替、スマートダークモード（Gemini/Gemini Enterprise対応）

🌐 **言語**: [简体中文](README.md) | [English](README_EN.md) | **日本語** | [한국어](README_KO.md) | [Deutsch](README_DE.md) | [Français](README_FR.md) | [Español](README_ES.md) | [Português](README_PT.md) | [Русский](README_RU.md)

## ✨ 機能

### 📝 プロンプト管理

- **クイック挿入**：よく使うプロンプトをワンクリックでチャットに挿入
- **カテゴリ管理**：カテゴリのフィルタ、名前変更、削除に対応
- **検索機能**：必要なプロンプトを素早く検索
- **CRUD操作**：プロンプトライブラリをカスタマイズ・管理
- **コピー機能**：プロンプト内容をワンクリックでクリップボードにコピー
- **ドラッグ＆ソート**：プロンプトの表示順を自由に調整

### 📁 会話管理

- **フォルダアーカイブ**：カスタムフォルダを作成してチャット履歴を整理
- **マルチカラータグ**：30種以上の伝統的な中国色、カスタムカラーとマルチタグ管理に対応
- **リアルタイム検索**：タイトルで素早くフィルタ、タグ組み合わせフィルタに対応
- **一括操作**：一括削除、移動、アーカイブのための複数選択
- **会話エクスポート**：Markdown/JSON/TXT形式でエクスポート、画像はBase64に変換可能
- **シームレス同期**：Geminiサイドバーの最新データを自動同期（標準版/エンタープライズ対応）

### 📑 アウトラインナビゲーション

- **自動抽出**：AIレスポンスから見出し構造を抽出（標準版とエンタープライズShadow DOM対応）
- **ユーザークエリグループ**：会話ターンごとにアウトラインをグループ化、ユーザークエリをグループヘッダーに（💬アイコン）
- **スマートインデント**：最高レベルに基づいてインデントを自動調整し、左側の空白を削減
- **クイックジャンプ**：アウトライン項目をクリックでスムーススクロール、2秒間ハイライト
- **同期スクロール**：ページスクロール時に対応するアウトライン項目を自動ハイライト（設定で切替可能）
- **レベルフィルタ**：見出しレベル表示を設定、レベル0でユーザークエリのみに素早く折りたたみ
- **切替コントロール**：無効時にアウトラインタブを自動非表示

### 🚀 クイックナビゲーション

- **トップ/ボトムへジャンプ**：長い会話での素早い位置決め
- **フローティングボタングループ**：パネル折りたたみ時もアクセス可能

### 📐 ページ幅

- **カスタム幅**：ピクセル（px）とパーセント（%）の両方に対応
- **即時適用**：調整後すぐに適用、リフレッシュ不要
- **独立設定**：サイトごとに異なる設定が可能

### ⚓ スマート位置システム

2つの独立した位置記録システム：

- **閲覧履歴（リーディングプログレス）**：
  - 長期的な「閲覧進捗メモリ」、リフレッシュ/セッション間の復元に対応
  - スクロール時に自動記録、GM_storageに永続化
  - ページ読み込みや会話切り替え時に自動復元

- **双方向アンカー**：
  - 短期的な「戻りポイント」、ブラウザの戻るや`git switch -`に類似
  - アウトライン/トップ/ボトムボタンクリック時に現在位置を自動保存
  - 2つの位置間の往復切り替えに対応

### 🏷️ タブ拡張

- **生成ステータス表示**：タブタイトルに⏳（生成中）または✅（完了）ステータスアイコンを自動表示
- **カスタムタイトル形式**：`{status}{title}[{model}]`プレースホルダーの組み合わせに対応
- **プライバシーモード（ボスキー）**：タブタイトルを「Google」に偽装、会話内容を非表示
- **完了通知**：バックグラウンド生成完了時にデスクトップ通知を送信
- **ウィンドウ自動フォーカス**：生成完了時にブラウザウィンドウを自動で前面へ

### ⚙️ 設定パネル

- **タブ切替**：プロンプト、アウトライン、設定の3つのタブ
- **パネル設定**：デフォルトの展開/折りたたみをカスタマイズ、外側クリックで自動非表示
- **中国語入力修正**：エンタープライズでの最初の文字の問題を修正するオプショントグル
- **言語切替**：簡体字中国語/繁体字中国語/英語に対応

### 🎯 スマート適応

- ✅ Gemini 標準版 (gemini.google.com)
- ✅ Gemini エンタープライズ (business.gemini.google)

### 🌓 自動ダークモード

- **スマート検出**：システム/ページのライト/ダークモード切替にリアルタイムで追従
- **フル適応**：丁寧に調整されたダークテーマカラースキーム、目に優しい

### 📋 コンテンツアシスト

- **数式ダブルクリックコピー**：数式をダブルクリックでLaTeXソースをコピー、デリミタを自動追加
- **テーブルMarkdownコピー**：テーブル右上にコピーボタンを追加、Markdown形式を直接コピー
- **透かし除去**：Gemini AI生成画像からNanoBanana透かしを自動除去
- **エッジスナップ非表示**：パネルを画面端にドラッグすると自動非表示、ホバーで表示
- **手動アンカー**：クイックツールバーでアンカー位置の設定/戻り/クリア

## 📸 プレビュー

- 右側にフローティングパネル、ドラッグ＆移動に対応（最適化された体験、誤ったテキスト選択なし）
- グラデーションテーマ、美しい外観
- フローティングバーに現在のプロンプトを表示、ワンクリッククリアに対応

## 🔧 使い方

1. Tampermonkeyブラウザ拡張をインストール
2. このスクリプトをインストール
3. Geminiページを開くと、右側にプロンプト管理パネルが表示されます
4. プロンプトをクリックして素早く挿入

## ⌨️ クイック操作

| 操作 | 説明 |
| --- | --- |
| プロンプトをクリック | 入力ボックスに挿入 |
| 📋 コピーボタン | プロンプト内容をコピー |
| ☰ ドラッグハンドル | ドラッグして順序を調整 |
| ✏ 編集ボタン | プロンプトを編集 |
| 🗑 削除ボタン | プロンプトを削除 |
| ⚙ カテゴリ管理 | カテゴリの名前変更/削除 |
| ×ボタンをクリック | 挿入したコンテンツをクリア |
| Enterで送信 | フローティングバーを自動非表示 |
| ⬆ / ⬇ ボタン | ページのトップ/ボトムへジャンプ |
